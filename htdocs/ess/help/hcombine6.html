Travel with Lodging Wizard

This screen provides a single form with which to create a simple expense report or to enter information about a trip within an expense report.  Once you've created entries on a report with this screen, the entries, and report can be treated like any other report.   It is important that after entering   information into this form that you click the following button at the bottom of the screen:

Finish Wizard - add these items to report (button) - This button will add the contents of the screen to the expense report.

En lieu of selecting the "Finish Wizard..." button you can click on the "Click to add another purpose" link to continue with another trip screen or select the "Click to add another expense" link to add more expenses to the purpose on this screen.

For field related information for each screen section, see the help for each the following screens:

  • Trip Purpose - Single Purpose help.

  • Lodging - Hotel Receipt help.

  • Mileage - Mileage Claim help.

  • Expense Receipts - Basic Receipt help.
