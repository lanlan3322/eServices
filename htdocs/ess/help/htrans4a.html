Odometer Reading

This screen is used to enter an odometer reading report for a fleet vehicle.  Although you won't get reimbursed from the entry of this screen, it is required if you have a fleet vehicle.

Week Ending: - (required) Enter the date to indicate which week the odometer reading covers.  The date should be a Saturday and you should only file one odometer reading report for any one week.

Vehicle No.: - Enter your vehicle number.  The system will remember and populate this date from session to session.

Date Put In Service: - Enter your service date.  The system will remember and populate this date from session to session.

Start: - Enter the beginning odometer reading for the period.   The system will populate the previous finish reading in here when the screen is loaded.

Finish: - Enter the odometer reading as of the week ending date.

Total: - (protected) This is the difference between the finish and the start odometer readings.

Business: - Enter the number of miles that the car was used for business purposes.

The screen also contains the following links and buttons to assist in entering information:

calendar - This link will pop-up a calendar.  Any date that you select by clicking on it will be entered into the associated date field.  The single arrows will move the calendar one month while the double arrows move the calendar a year.

Update report with this vehicle information (button) - (required) After you have entered all the information required to report your odometer reading, click this button to add this information to the report you are creating or editing.  If you exit this screen without using this button, the information you provided will not be added to the report and will be lost.

Return to report display (link) - Use this link if you wish to abandon data entry and return to the report display without saving any of the information you may have added or changed.
