Mileage Claim

This screen is used to enter a mileage claim for vehicle usage reimbursement.  You will be reimbursed at an agreed upon rate.  These are the screen's fields:

Date: - (required) Enter the date travel took place.

Distance: - (required) Enter the distance traveled.  This label will actually be "Miles" or "Kilometers"

Rate: - (required) This rate is pre-populated for you.  If you feel that this rate is incorrect, you can change it.  If you change the rate it will most likely cause the report to fail the guideline check.

Amount: - (protected) This field displays the amount you will be reimbursed.  This field is calculated and is based upon the number of miles entered times the mileage reimbursement rate.

Purpose: (list) - (required) Select the purpose form your report to which you want to tie this expense.

Billable: (list)  - If this field is present, select either 'Yes' if the expense is billable to the client or project specified in the purpose or 'No' if not billable.  Some installations do not include this field.

From: - (required) Enter the starting point of your travel.   This may be your home or your office.

To: - (required) Enter the destination for your trip.  This will normally be your clients site, meeting place, etc.  For a round-trip entry, you may end the destination with 'RT' and include all miles traveled in one entry.

The screen also contains the following links and buttons to assist in entering information:

calendar - This link will pop-up a calendar.  Any date that you select by clicking on it will be entered into the associated date field.  The single arrows will move the calendar one month while the double arrows move the calendar a year.

Update report with this mileage claim (button) - (required) After you have entered all the information required to describe this receipt-expense, click this button to add this information to the report you are creating or editing.  If you exit this screen without using this button, the information you provided will not be added to the report and will be lost.

Return to report display (link) - Use this link if you wish to abandon data entry and return to the report display without saving any of the information you may have added or changed.
