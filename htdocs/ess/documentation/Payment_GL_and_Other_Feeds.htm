After reports have been added to the end-of-day batch, they can be used to create a feed file.  Feed files are used to update other systems.  The Expense Submittal System (ESS) comes standard with three feeds, but can easily be updated with a customized feed.  The three feeds that ESS comes with are:

 

  • Generic Payments – This feed contains information on all payments. 
  • Generic G/L Feed – This feed contains entries for the general ledger.
  • NACHA Feed – This is a file that can be submitted to your bank for payment via the ACH system (electronic checks).
