Changing Screen Text and Messages

March 28, 2006

The screens and messages that the Expense Submittal System (ESS) uses to communicate with users can be costumized.  Concerning screen text and messages, they can be broken down into two groups

  • Some screen text and messages can only be customized by modifying the HTML and JavaScript.
  • Other screen text and message can be customized by changing the ESS configuration files.  Specifically, the affected configuration files are: system.xml, Status.xml and rules.xml.

 

This document deals with the second group of screen text and messages.
