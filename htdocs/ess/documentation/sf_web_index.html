Expense Submittal System™ Project

Official Home Page
The Expense Submittal System™ (ESS) is a soup-to-nuts solution for corporate expense reporting needs.  ESS is Web-based and is programmed in JSP, Java, JavaScript and HTML.  It is currently run in production under Linux, Windows and Solaris.  Functionally it handles: report creation, approval routing, approvals, policy enforcement via guidelines, auditing, receipting, payments, posting to your G/L, and investigations.  ESS is licensed under the GNU Public License.

ESS can be installed in an existing Web application server (e.g., Tomcat) - database (e.g., MySQL) environment.
Source Forge - The ESS project is maintained on Source Forge.  To participate in the project or download a copy of ESS, go to the ESS project page at http://www.sourceforge.org/projects/expense-ss.
 
ExpenseServices.com - We provide support and services for ESS.  To access ESS's Web site where you can download the software, access documentation and find out more about various services, go to http://www.expenseservices.com. Registration is required for some part of the Web site, but it is free.
