Shutdown Message Generator

In the field below, enter a short text message to inform users of a pending system shutdown.  This message will apear in an alert box on all logged-in expense users' browser during the next one-half hour.  When indicating shutdown time, indicate an absolute time (e.g., 10:30 AM Central Time) and not a relative time (e.g., thirty minutes) as the message will take time to populate to all users.  'Multiple view' indicates that the user will receive multiple alert notices if they remain on the system.  'Single view' indicates that the user will receive the notice just once.  To turn off an existing message, enter a blank text message.
